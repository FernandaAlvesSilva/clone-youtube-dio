# 📺 Clone do YouTube

Projeto desenvolvido como desafio da Digital Innovation One (DIO), consistindo na criação de um clone simples do YouTube. O foco foi reproduzir a tela inicial e a tela de exibição de vídeo, sem aplicar técnicas de responsividade.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## 📄 Funcionalidades

- Página inicial com layout semelhante ao YouTube:
  - Barra lateral de navegação
  - Área de vídeos em miniatura
  
- Página de exibição de vídeo:
  - Vídeo principal
  - Informações do vídeo (título, descrição, etc.)
  - Lista de vídeos sugeridos na lateral
